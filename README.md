# 🌍 World Atlas

A React app that fetches live data from the REST Countries API.

## Features
- Fetches 250+ countries from a public API
- Search by country name or capital
- Filter by region
- Sort by name, population, or area
- Click any card for detailed view
- Handles loading and error states
- Fully responsive

## Tech Stack
- React + Vite
- REST Countries API (no API key needed)
